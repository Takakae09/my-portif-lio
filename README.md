# 🌐 Portfólio - Caio Luan

Este é o meu portfólio pessoal como desenvolvedor Front-End, criado com HTML, CSS e JavaScript.

O objetivo do projeto é apresentar minhas habilidades, projetos e evolução como desenvolvedor.

---

## 🚀 Demonstração

📌 Você pode acessar o projeto abrindo o arquivo `index.html` no navegador.

---

## 🧠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Particles.js  
- Font Awesome  

---

## 🎨 Funcionalidades

- Layout responsivo  
- Animação de partículas no fundo  
- Seção de apresentação pessoal  
- Lista de habilidades (Skills)  
- Área de projetos  
- Formulário de contato  
- Botão de tema (Dark/Light mode 🌙☀️)  
- Scroll suave entre seções  

---

## 🌙 Tema (Dark / Light)

O projeto possui um sistema de troca de tema:

- Modo escuro padrão
- Modo claro ativado por botão
- Estilização baseada em variáveis CSS
